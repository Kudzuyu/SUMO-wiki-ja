TraCI/車線に関する値の取得¶

コマンド 0xa3: 車線変数の取得¶

ubyte string
変数 Lane ID

名前の着いた車線に関する値を取得します。 以下の変数の値を取得することができます。 返り値の型も表内に示されています。

車線変数取得の概要
変数 型 説明 [Python メソッド]
id list (0x00) stringList シナリオ中の全車線の ID リスト (与えられた車線 ID は無視される) getIDList
count (0x01) int シナリオ中の全車線の数 (与えられた車線 ID は無視される) getIDCount
link number (0x30) ubyte 車線から流出するリンクの数 [#] getLinkNumber
edge id (0x31) string getEdgeID
links (0x33) compound getLinks
allowed vehicle classes (0x34) stringList getAllowed
disallowed vehicle classes (0x35) stringList getDisallowed
length (0x44) double getLength
vmax (0x41) double getMaxSpeed
shape(0x4e) shape getShape
width (0x4d) double getWidth
CO2 emissions (id 0x60) double getCO2Emission
CO emissions (id 0x61) double getCOEmission
HC emissions (id 0x62) double getHCEmission
PMx emissions (id 0x63) double getPMxEmission
NOx emissions (id 0x64) double getNOxEmission
fuel consumption (id 0x65) double getFuelConsumption
noise emission (id 0x66) double getNoiseEmission
electricity consumption (id 0x71)

Response 0xb3: 車線変数¶

ubyte string ubyte
変数 車線 ID 変数の型

"車線変数取得コマンド"への応答。

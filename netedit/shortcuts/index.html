    NETEDIT view shortcuts¶

    In addition to the standard sumo-gui shortcuts for moving the view there is a large number of netedit-specific shortcuts.

    Shortcut
    Ctrl + G Toggle grid
    Ctrl + PgUp Increase grid size
    Ctrl + PgDown Shrink grid size
